Search filter

lx-search-filter is a directive that create a search filter with various style options.

This component has been completly reworked since LumX beta version (v0.3.96).
Please refer to the updated documentation.

Parameter Type Default Description
lx-closed boolean false Define if the input is hidden and should be opened on search icon click.
lx-color string black Define the search filter color. Options: colors defined in colors and sizes section in dist/scss/core/settings/defaults.scss
lx-width string 240 Define the search filter width in pixels. A percent value can be defined too. The default value is applied only in closed mode.
